Politique de confidentialité

Cette politique de confidentialité décrit comment nous collectons, utilisons et protégeons les données personnelles que vous nous fournissez via notre formulaire de contact.

1. Données collectées

Nous collectons uniquement les informations que vous choisissez de nous communiquer, notamment :

  • Votre nom
  • Votre numéro de téléphone
  • Votre adresse (facultatif)
  • Votre message

2. Utilisation des données

Ces données sont utilisées exclusivement pour :

  • Organiser votre rendez-vous de coiffure à domicile
  • Vous contacter pour toute information liée à votre demande

3. Conservation des données et utilisation

Les données que vous saisissez dans le formulaire ne sont pas stockées ni conservées par notre site. Elles sont uniquement utilisées temporairement dans votre navigateur pour générer le contenu du SMS que vous envoyez via votre application de messagerie. Aucun transfert ou enregistrement de ces données n’est effectué sur un serveur.
